Complexity Ratio: 8x

Gen's interactive UI is 8 times larger than Bub's. The extra complexity comes from: streaming syntax highlighting, adaptive render engine, multi-picker system, extension UI bridge, diff renderer, and 3-layer state management (StateManager + RenderEngine + CommitManager).
